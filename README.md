## Features

### Items Management
- Add new items with **name**, **category**, and **price**.
- View all items in a neat table.
- Search items by name.
- Update item details.
- Delete items.

### Employees Management
- Add new employees with **name**, **post**, and **salary**.
- View all employees.
- Search employees by name.
- Update employee details.
- Delete employees.

### General
- Simple and intuitive menu system.
- Input validation for numeric fields (`price` and `salary`).
- Uses SQLite database (`grocery.db`) for data storage.
- Compatible with Windows CMD, Linux, or Mac terminals.
